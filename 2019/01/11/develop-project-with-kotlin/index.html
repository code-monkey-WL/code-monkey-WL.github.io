使用 kotlin 开发项目的体验

Posted on 2019-01-11 In kotlin

自从 Google 宣布 Kotlin 成为 Android 官方开发语言后，Kotlin 着实火了一阵子，我也去学习了一下它的语法，一眼望去都是满满的语法糖啊，怪不得有人说最好的 java 第三方库是 kotlin……，在熟悉了 kotlin 的语法后，写了个公司的内部项目，不得不说，体验非常好。

不仅如此，Kotlin 是空安全的，它在编译期间就会处理各种为null的情况，很大程度上避免出现 NullPointException，而且使用 var 和 val 定义变量也非常简介，甚至可以省略分号，在写多了 kotlin 之后，写 java 我是老是忘记分号的存在。

整个项目使用 kotlin 写下来几乎没有遇到什么坑，我对kotlin的评价非常高，是我目前接触的语言中用的比较舒服的了，避免了 java 臃肿的语法，同时又完全兼容 java，以后做一些个人项目的话，应该就用它啦。
