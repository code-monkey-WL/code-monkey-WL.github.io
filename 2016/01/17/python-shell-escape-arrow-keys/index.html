python shell 方向键乱码

Posted on 2016-01-17 In python

最近在服务器上安装了 python3 的环境，安装过程记录如下：


在安装完成后，输入 python3 进入终端环境后出现了一个奇怪的问题，当我点击方向键移动光标的时候，出现的是转义字符:

为什么会出现这种情况呢，我搜索了一下这个问题，大致总结如下：

我们使用的终端模拟器，都是用软件来模拟当年硬件实现的终端，并且还在底层帮你把按下的方向键转化为能够用来移动光标的特定转义字串，这个功能需要用到一个库，就是 The GNU Readline Library。但问题在于，readline 采用 GPL v3 授权，这一授权要求所有使用 readline 的程序必须开源，所以我的 centos 服务器上是没有这个库的，那么我编译的 python shell 也就没有这个功能。

两种解决方法解决方法：

  1. 安装 readline 重新编译 yum install readline-devel
  2. pip install readline
