spring boot properties中文乱码

Posted on 2016-02-19 In java

前言

最近在用spring boot框架做项目的时候，使用hibernate-validation校验的时候的时候，配置在ValidationMessages.properties里面的中文的时候出现了乱码现象，在此记录一下解决过程，以备忘。

解决过程

spring-boot会默认使用ValidationMessages.properties作为解析错误信息的文件。
首先我觉得应该是properties文件自身的编码问题，因为我使用的是Intellij IDEA开发,它默认的properties文件的编码是系统默认编码，在我的window上是GBK编码，所以首先修改了properties文件的编码类型，选择File->setting->File Encoding选择properties文件，修改编码为UTF-8但是仍未解决乱码问题。
后来大致阅读了部分源码才发现了问题所在:
